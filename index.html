Qui suis-je ?

Je suis développeur web depuis 5 ans. J'ai construit des sites e-commerce avec Magento, des API REST avec Symfony2 et des webviews mobiles avec AngularJS.

Je suis maintenant consultant web indépendant et je souhaite aider les entreprises à résoudre les problèmes qu'elles rencontrent avec leur applicatif web.
